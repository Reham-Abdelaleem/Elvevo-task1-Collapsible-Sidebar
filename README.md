# 📱 Elvevo Task 1 - Collapsible Sidebar

A sleek, minimal, and responsive collapsible sidebar built as part of the **Elevvo Training Program**. This project demonstrates a clean UI/UX with smooth transitions, leveraging **Bootstrap 5** for layout and **Vanilla JavaScript** for functionality.

## 📝 Description
Leon - Creative Digital Agency Template inspired sidebar. This task focuses on creating a dynamic sidebar section that toggles open and closed with smooth animations. It includes a logo placeholder, navigation links with icons, and a fully responsive design that works on all screen sizes.

## ✨ Features
* **Fully Responsive Design:** Adapts perfectly to desktops, tablets, and mobile phones.
* **Smooth Toggle Animation:** Uses CSS3 transitions for a premium feel when opening/closing.
* **Logo Placeholder:** Dedicated section at the top for branding.
* **Navigation List:** Includes 5 interactive links with professional icons.
* **Built with Bootstrap 5:** For a structured, modern, and clean grid layout.
* **Vanilla JavaScript:** Light-weight and fast DOM manipulation without external libraries.

## 🛠️ Technologies Used
* **HTML5 & CSS3** (Semantic structure and custom styling)
* **Bootstrap 5** (Layout and responsiveness)
* **Vanilla JavaScript** (Toggle functionality)
* **Font Awesome** (Modern navigation icons)
* **Google Fonts** (Work Sans)
