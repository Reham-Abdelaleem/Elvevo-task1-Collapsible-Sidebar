# Elvevo Task 1 - Collapsible Sidebar

## Description
A modern, minimal, and responsive collapsible sidebar built as part of the Elvevo training program. This project demonstrates a clean UI/UX with smooth animations, leveraging **Bootstrap 5** for layout and responsiveness.

## Features
* **Fully responsive design** (Bootstrap Grid & Breakpoints)
* **Smooth toggle animation** for opening and closing
* **Placeholder for Logo** at the top
* **Navigation list** (4-5 links with icons)
* **Toggle functionality** handled with Vanilla JavaScript
* **Built with Bootstrap 5** for a professional look

## Technologies
* **HTML5** & **CSS3**
* **Bootstrap 5** (Layout & Components)
* **JavaScript** (DOM Manipulation)
* **Font Awesome** (For Sidebar Icons)
* **Google Fonts** (Work Sans)
